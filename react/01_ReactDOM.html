# SPA (Single Page Application)

  • 브라우저가 ajax를 통해 아주 작은 데이터를 요청해서 가져올 수 있게 되었기 때문에 전체 웹 애플리케이션을 한 페이지로 만들 수 있게 되었다
  • 사용자는 사이트를 이동하는 것처럼 느끼지만 실제로는 같은 페이지 안에 머물도록 만들 수 있다

# React

  • 브라우저 DOM을 갱신하면서 사용하기 위한 라이브러리
  • 리액트가 모든 처리를 대신 해주기 때문에 더 이상 페이지를 효율적으로 만들기 위해 여러 복잡한 내용을 신경쓸 필요가 없어진다
  • 리액트에서는 코드로 DOM을 직접 조작하지 않는다
  • DOM을 직접 조작하는 대신 어떤 UI를 생성할지 리액트에게 지시한다
  • HTML Element와 비슷하지만 다른 React Element를 사용한다
